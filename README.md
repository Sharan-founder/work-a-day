# Work A Day

Work A Day is an AI-powered platform designed to connect daily wage workers with job opportunities in real-time.

## Problem
Daily wage workers often struggle to find jobs quickly, while contractors face difficulty finding reliable workers. The current system depends heavily on middlemen and manual searching.

## Solution
Work A Day enables:
- Voice-based skill input (local language support)
- AI-based skill detection
- Instant job matching
- Trust and rating system

## Features
- Voice onboarding for workers
- Smart job matching
- Contractor job posting
- Ratings & trust system

## Tech Stack
- Flutter (Frontend)
- Node.js / Python (Backend)
- MongoDB / PostgreSQL (Database)
- Firebase / AWS (Cloud)
- Google Speech-to-Text API
